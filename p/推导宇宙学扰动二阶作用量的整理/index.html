推导宇宙学扰动二阶作用量的整理

用AMD3+1分解形式度规，变分得到能量约束方程和动量约束方程，再将主要的曲率量做微扰展开，代入原始作用量中，展开并整理得到二阶微扰作用量 .

Mar 28, 2026

前言

最近在整理一些inflation的内容，虽然本科做了一点原初扰动有关的工作，但其实一直没有对二阶的微扰作用量进行一个完整的推导，这里整理了一下推导的思路和部分结论，完整的推导实在太冗长了，还是放在草稿纸上吧 。。。

基本定义

ADM度规为

$$ \mathrm{d}s^{2 } = -N^{2 }\mathrm{d}t^{2 } + h _{ij}\left( N^{i }\mathrm{d}t + \mathrm{d}x^{i } \right)\left( N^{j }\mathrm{d}t + \mathrm{d} x^{j } \right) \,, $$

其中

$$ g _{\mu \nu} = \begin{pmatrix} -N^{2 }+N^{k }N _{k} & N _{i} \\ N _{j} & h _{ij} \end{pmatrix} \,,\quad g^{\mu \nu } = \begin{pmatrix} -1 / N ^{2 } & N ^{i } / N^{2 } \\ N ^{j } / N ^{2 } & h ^{ij } - N^{i }N^{j } / N^{2 } \end{pmatrix}\,. $$

超曲面上的法矢量

$$ n _{\mu} = (-N,0,0,0)\,,\quad n ^{\mu } = (1 / N, - N^{i } / N)\,. $$

外曲率为

$$ K _{\mu \nu} = \frac{1}{2}\mathcal{L} _{n}(g _{\mu \nu} + n _{\mu} n _{\nu})\,, $$

三维曲率（不考虑全导数项）为

$$ R = \ ^{(3) }R - K^{2 } + K^{\mu } _{\;\nu} K^{\nu } _{\;\mu} \,, $$

在幺正规范 $\delta \phi = 0$ 下，作用量为

$$ \begin{aligned} S &= \int \mathrm{d}^{4 }x \sqrt{ -g }\left[ \frac{M _{p}^{2 }}{2}R- \frac{1}{2} g ^{\mu \nu } \partial _{\mu} \phi \partial _{\nu}\phi -V(\phi) \right] \\ & = \frac{M _{p} ^{2 }}{2} \int \mathrm{d} ^{4 }x \; N \sqrt{ h }\left( \ ^{(3) } R + K _{ij} K ^{ij } - K ^{2 } \right) \\ &\quad\quad + \int \mathrm{d } ^{4 }x N\sqrt{ h }\left[ \frac{1}{2N^{2 }}\left( \dot{\phi} - N^{i }\partial _{i}\phi \right)^{2 }- \frac{1}{2}h^{ij }\partial _{i}\phi \partial _{j}\phi-V(\phi) \right] \\[6pt] & =\frac{M _{p} ^{2 }}{2}\int \mathrm{d} ^{4 } x\; N\sqrt{ h }\left( \ ^{(3) }R + K _{ij}K^{ij } - K^{2 } \right) \\ &\quad\quad+\int \mathrm{d }^{4 }x N\sqrt{ h }\left[ \frac{1}{2N^{2 }}\dot{\phi} ^{2 }-V(\phi) \right]\,, \end{aligned} $$

此外，时移函数和位移函数为

$$ N = 1+\alpha\,,\quad N _{i} = \partial _{i}\beta\,. $$

其中 $\alpha,\beta$ 都是一阶小量 .

变分得到能量约束方程和动量约束方程

  • 对 $N$ 变分得到能量约束方程

其中 $K _{ij}= \frac{1}{2N}\left( \dot{h} _{ij} - D _{i}N _{j}-D _{j}N _{i} \right)$ ，定义

$$ E _{ij} = NK _{ij} = \frac{1}{2}\left( \dot{h} _{ij} - D _{i}N _{j}-D _{j}N _{i} \right)\,, $$

把作用量中的 $K _{ij}$ 和 $K$ 全部用 $E _{ij}$ 和 $E$ 表示，对 $N$ 变分可以得到能量约束方程：

$$ \frac{M _{p}^{2 }}{2} \left[ \ ^{(3) }R - \left(K _{ij} K^{ij }-K^{2 } \right) \right] = \left( \frac{1}{2N^{2 }}\dot{\phi}^{2 } + V(\phi) \right) \,.\tag{$\star$} $$
  • 对 $N^{i }$ 变分得到（取 $\partial _{i}\phi = 0$ ）动量约束方程

对于 $S _{\phi}$ 部分由于选取的是幺正规范，因此

$$ \frac{\delta S _{\phi}}{\delta N _{i}}=0\,, $$

对于引力部分

$$ \delta \left( K _{ij}K^{ij }-K^{2 } \right) = 2(K^{ij }-h^{ij }K)\left( - \frac{1}{2N} \right)\left[ D _{i}\delta N _{j}+D _{j}\delta N _{i} \right] $$

注意括号内部的$K^{ij }-h^{ij }K$ 关于指标 $i,j$ 对称，因此通过分部积分可以得到动量约束方程

$$ D _{i}\left( K^{ij }-h ^{ij }K \right) = 0\,.\tag{$\star$} $$

对 $ \ ^{(3) }R,K,K _{ij}$ 的微扰展开

度规展开

$$ \begin{aligned} h _{ij} &= a^{2 }\mathrm{e}^{ 2\zeta }\delta _{ij}\simeq a^{2 }\left( 1+2\zeta + 2\zeta ^{2 } \right) \delta _{ij} \\ h^{ij} &= a^{-2 }\mathrm{e}^{ -2\zeta }\delta _{ij}\simeq a^{-2 }\left( 1-2\zeta + 2\zeta ^{2 } \right) \delta _{ij} \end{aligned} $$

联络为

$$ \begin{aligned} \Gamma ^{i } _{jk} &=h^{il}\left( \partial _{j}h _{lk}+\partial _{k}h _{jl}-\partial _{l}h _{jk} \right) \\ & \simeq \delta ^{i } _{\;j}\partial _{k}\zeta + \delta ^{i } _{\;k} \partial _{j}\zeta - \delta _{jk}\partial ^{i }\zeta\,. \end{aligned} $$
  • $ \ ^{(3) }R$ 的微扰展开为
$$ \begin{aligned} \ ^{(3) }R &= h ^{ij }\left( \partial _{k}\Gamma ^{k } _{ij} - \partial _{j}\Gamma ^{k } _{ik} + \Gamma ^{k } _{kl}\Gamma ^{l } _{ij} - \Gamma ^{l } _{ik}\Gamma ^{k } _{jl} \right) \\ &= a^{-2 }\mathrm{e}^{ -2\zeta }\left[ -4\partial ^{2 }\zeta - 2\left( \partial \zeta \right)^{2 } \right] \\ &\simeq -\frac{4}{a^{2 }}\left[ \partial ^{2 }\zeta -2\zeta \partial ^{2 }\zeta + \frac{1}{2} \left( \partial \zeta \right)^{2 } \right] \,. \end{aligned} \tag{$\star$} $$
  • $K _{ij}$ 的微扰展开：
$$ \begin{aligned} K _{ij} &= \frac{1}{2N}(\dot{h} _{ij} - D _{i}N _{j}-D _{j}N _{i}) \\ & = \frac{1}{1+\alpha}\left[ (H+\dot{\zeta})h _{ij} - \partial _{i}\partial _{j}\beta + \Gamma ^{k } _{ij}\partial _{k}\beta \right] \,, \\ \implies\qquad& \\ K _{ij}^{(0) } &= a ^{2 }H \delta _{ij}\,, \\[6pt] K^{(1) } _{ij} & = a^{2 }H\left( 2\zeta - \alpha \right)\delta _{ij} + a^{2 }\dot{\zeta}\delta _{ij} - \partial _{i}\partial _{j}\beta\,, \\[6pt] K^{(2) } _{ij} &= a^{2 }H\left( 2\zeta - 2\alpha \zeta + \alpha ^{2 }\right) \delta _{ij} + a^{2 }\dot{\zeta}\left( 2\zeta-\alpha \right)\delta _{ij} + \alpha \partial _{i}\partial _{j}\beta \\ & \quad +\partial _{i}\zeta \partial _{j}\beta + \partial _{j}\zeta \partial _{i}\beta - \delta _{ij}\partial _{k}\zeta \partial ^{k }\beta\,. \end{aligned} $$

根据 $K^{i } _{;j}=h^{ik }K _{kj},h^{ik }\simeq \frac{1}{a^{2 }}\left( 1-2\zeta \right)\delta ^{ik }$ ，有

$$ \begin{aligned} K^{i } _{\;j} &\simeq H\delta ^{i } _{\;j} - H\alpha \delta ^{i } _{\;j} + \dot{\zeta} \delta ^{i } _{j}- \frac{1}{a^{2 }}\partial ^{i }\partial _{j}\beta\,. \\ K &\simeq 3H + 3\left( \dot{\zeta}-H\alpha \right) - \frac{\partial^{2 }\beta}{a^{2 }}\,. \end{aligned} $$
  • $K$ 的微扰展开（和上面的方法结果一致）：
$$ \begin{aligned} K &= h^{ij }K _{ij } = h ^{ij }\cdot \frac{1}{2N}\cdot \left[ 2(H+\dot{\zeta})h _{ij}-2D^{j }N _{j}\right] \\ &= \frac{1}{N}\left[ 3(H+\dot{\zeta})-D^{j }N _{j}\right]\,, \\ \implies \qquad& \\ K^{(0) } & = 3H\,, \\[6pt] K^{{(1)} } & = 3 \left( \dot{\zeta} - H\alpha \right) - \frac{1}{a^{2 }}\partial ^{2 }\beta\,, \\ K^{(2) } &= 3 \left( H\alpha ^{2 }-\alpha \dot{\zeta} \right) + \frac{2 \zeta + \alpha }{a^{2 }}\partial ^{2 }\beta - \frac{1}{a^{2 }}\partial _{i}\zeta \partial ^{i }\beta\,. \end{aligned} $$

把展开后的 $ \ ^{(3) }R,K,K _{ij}$ 代入到约束方程中，解出 $\alpha,\beta$ .

  • 把微扰项带入到动量约束方程 $D _{i}\left( K^{i } _{;j}-\delta ^{i } _{;j}K \right)=0$ ，这里注意背景层面 $K^{{0}i } _{;;j}-K\delta ^{i } _{;j}\propto \delta ^{i } _{;j}$ ，因为
$$ D _{i}\delta ^{i } _{\;j} = \partial _{i}\delta ^{i } _{\;j}\,, $$

在线性阶、平直 FLRW 背景上，协变导数与普通导数的差别只会产生更高阶项，所以可以把协变导数转化为普通导数，即

$$ \begin{aligned} D _{i}\left( K^{i } _{\;j}-\delta ^{i } _{\;j}K \right)=0 \Leftrightarrow \partial _{i}\left( K^{i } _{\;j}-\delta ^{i } _{\;j}K \right)=0 \\ \implies \partial _{i}\left( K^{(1) i} _{\quad j}-\delta ^{i } _{\;j}K^{(1) } \right) =0\,. \end{aligned} $$

最终有

$$ \alpha = \frac{\dot{\zeta}}{H}\,. \tag{$\star$} $$
  • 把微扰项带入到能量约束方程 $\frac{M _{p}^{2 }}{2} \left[ \ ^{(3) }R - \left(K _{ij} K^{ij }-K^{2 } \right) \right] = \left( \frac{1}{2N^{2 }}\dot{\phi}^{2 } + V(\phi) \right)$ ，展开到一阶，左边为
$$ LHS \simeq 2M _{p}^{2 }\left[ - \frac{1}{a^{2 }}\partial ^{2 }\zeta + 3 H \dot{\zeta} - 3H^{2 } \alpha - H \frac{\partial^{2 }\beta}{a^{2 }} \right] \,, $$

右边

$$ RHS = \frac{1}{2(1+\alpha)^{2 }}\dot{\phi}^{2 } + V(\phi) \simeq \frac{1}{2}\left( 1-2\alpha \right)\dot{\phi}^{2 }+V(\phi)\,, $$

一阶项只有 $-\alpha \dot{\phi}^{2 }$ ，再根据背景方程 $\dot{\phi}^{2 } = -2M _{p}^{2 }\dot{H}$，最终有

$$ \partial ^{2 }\beta = - \frac{1}{H}\partial ^{2 }\zeta + \epsilon _{H}a^{2 }\dot{\zeta}\,.\tag{$\star$} $$

把展开后的 $ \ ^{(3) }R,K,K _{ij}$ 代入到作用量中

  • 标量场

这里需要利用背景方程

$$ \begin{aligned} \dot{H} = -4\pi G(\rho + p) &\implies \dot{\phi} ^{2 }= -2M _{p}^{2 } \dot{H} \,,\\ -M _{p}^{2 }\left( 2\dot{H}+3H^{2 } \right) = \dot{\phi}^{2 }-\left( \frac{1}{2}\dot{\phi}^{2 } + V \right)&\implies V=M _{p}^{2 }\left( 3H^{2 } + \dot{H}\right) \,. \end{aligned} $$

体积元展开

$$ \sqrt{ h }=a^{3 }\mathrm{e}^{ 3\zeta } \simeq a^{3 }\left( 1+3\zeta + \frac{9}{2}\zeta ^{2 } \right) \,, $$

最后把作用量展开为，其中二阶项为

$$ \frac{\mathcal{L} _{\phi}^{(2) }}{M _{p}^{2 }} = a^{3 }\left[ -\dot{H}\alpha ^{2 } - 9H^{2 }{\color{Green} \alpha \zeta }- \left( \frac{27}{2}H^{2 }+9\dot{H} \right){\color{red} \zeta ^{2 } } \right] \,. \tag{$\star$} $$
  • 曲率项
$$ \begin{aligned} \mathcal{L} _{R} &= M _{p}^{2 }N\sqrt{ h } \ ^{(3) }R \\ &\simeq -2M _{p}^{2 }\left( 1+\alpha + 3\zeta \right) a \left[ \partial ^{2 }\zeta - 2\zeta \partial ^{2 }\zeta + \frac{1}{2}(\partial \zeta)^{2 } \right] \,, \end{aligned} $$

利用分部积分 $\zeta \partial ^{2 }\zeta\to -(\partial \zeta)^{2 }$ ，最终二阶项为

$$ \frac{\mathcal{L} _{ R}^{(2) }}{M _{p}^{2 }} = 2a\left( \frac{1}{2}(\partial \zeta) ^{2 } -\alpha \left( \partial ^{2 }\zeta \right) \right) \,. \tag{$\star$} $$
  • 外曲率动能项 外曲率动能部分 $\mathcal{L}_K$ 为 $$\mathcal{L}_K = \frac{M_p^2}{2} N\sqrt{h} (K _{ij}K^{ij} - K^2) = \frac{M_p^2}{2} \frac{\sqrt{h}}{N} (E _{ij}E^{ij} - E^2)$$

其中 $E^i _{;j} \equiv (H+\dot{\zeta})\delta^i _{;j} - a^{-2}\mathrm{e}^{-2\zeta}D^iD_j \beta$。对其作迹与平方的二阶展开，其中

$$ E _{ij}E^{ij } = E^{i } _{\;j}E^{j } _{\;i} = 3\left( H+\dot{\zeta} \right)^{2 } - 2 \left( H+\dot{\zeta} \right)a^{-2 }\mathrm{e}^{ -2\zeta } D^{2 }\beta + a^{-4 }\mathrm{e}^{ -4\zeta }\left( D^{i }D _{j}\beta \right)^{2 }\,, $$

以及

$$ E^{2 } = 9 \left( H+\dot{\zeta} \right)^{2 } - 6\left( H+\dot{\zeta} \right) a^{-2 }\mathrm{e}^{ -2\zeta } D^{2 }\beta + a^{-4 } \mathrm{e}^{ -4\zeta }\left( D^{2 }\beta \right) ^{2 }\,, $$

因此

$$ E _{ij}E^{ij }-E^{2 } = -6\left( H+\dot{\zeta} \right)^{2 } +4\left( H+\dot{\zeta} \right) a^{-2 }\mathrm{e}^{ -2\zeta } D^{2 }\beta + a^{-4 } \mathrm{e}^{ -4\zeta }\left[\left( D^{i }D _{j}\beta \right)^{2 } -\left( D^{2 }\beta \right) ^{2 } \right] \,, $$

同时

$$ \begin{aligned} D _{i}D _{j}\beta &=\partial _{i }\partial _{j}\beta - \partial _{j}\zeta \partial _{i}\beta - \partial _{i}\zeta \partial _{j}\beta + \delta _{ij}\partial ^{k } \zeta \partial _{k}\beta\,, \\ D^{2 }\beta &= \partial ^{2 }\beta + \partial ^{k }\zeta \partial _{k}\beta\,. \end{aligned} $$

代入 $E _{ij}E^{ij }-E^{2 }$ 中，其中 纯空间全导数项 $\partial_i\partial_j\beta \partial^i\partial^j\beta - (\partial^2\beta)^2 = \partial_i (\dots)$ 在全空间积分后直接为零，被舍弃，最终乘以体积因子 $\frac{\sqrt{h}}{N} \simeq a^3(1 - \alpha + \alpha^2 + 3\zeta - 3\alpha\zeta + \frac{9}{2}\zeta^2)$，提取纯二阶项

$$ \begin{aligned} \frac{\mathcal{L}_K^{(2)}}{M_p^2} =& -3H^2 a^3 \alpha^2 + 9H^2 a^3 \alpha\zeta - \frac{27}{2}H^2 a^3 \zeta^2 \\ & + 6H a^3 \alpha\dot{\zeta} - 2H a \alpha\partial^2\beta {\color{red} -18H a^3 \zeta\dot{\zeta} } + {\color{orange} 6H a \zeta\partial^2\beta } \\ & {\color{orange} - 4H a \zeta\partial^2\beta + 2H a (\partial_k\zeta)(\partial^k\beta) } - 3 a^3 \dot{\zeta}^2 + 2 a \dot{\zeta}\partial^2\beta + \text{边界项} \\[6pt] & = -3H^2 a^3 \alpha^2 + 9H^2 a^3 {\color{Green} \alpha\zeta } + 6H a^3 \alpha\dot{\zeta} - 2H a \alpha\partial^2\beta \\ &\qquad +{\color{red} \zeta ^{2 } }\left[ -\frac{27}{2}H^{2 }a^{3 } + 27 a^{3 }H^{2 } +9 \dot{H} \right] \\ &\qquad - 3 a^3 \dot{\zeta}^2 + 2 a \dot{\zeta}\partial^2\beta \end{aligned} \tag{$\star$} $$

其中

$$ {\color{red} - 18H a^3 \zeta\dot{\zeta} \to a^{3 }\left( 27H^{2 } + 9 \dot{H} \right)\zeta ^{2 }\,. } $$

将 $\mathcal{L}_\phi^{(2)}, \mathcal{L}_R^{(2)}, \mathcal{L}_K^{(2)}$ 相加，把 $\zeta ^{2 },\alpha \zeta$ 项抵消掉后，有

$$ \begin{aligned} \frac{\mathcal{L}^{(2)}}{M_p^2 } &= -a^3(\dot{H} + 3H^2)\alpha^2 + 6H a^3 \alpha\dot{\zeta}+ 2a(\underbrace{ \dot{\zeta} - H\alpha } _{ =0 })\partial^2\beta- 3a^3\dot{\zeta}^2 \\ &\qquad+ a(\partial\zeta)^2 - 2a\alpha\partial^2\zeta\\[6pt] &=-a^3(\dot{H} + 3H^2)\alpha^2 + 6H a^3 \alpha\dot{\zeta} - 3a^3\dot{\zeta}^2 + a(\partial\zeta)^2 - 2a\alpha\partial^2\zeta \\ &=a^3 (-\dot{H} - 3H^2) \frac{\dot{\zeta}^2}{H^2} + 6a^3 \dot{\zeta}^2 - 3a^3\dot{\zeta}^2 + a(\partial\zeta)^2 - 2a \frac{\dot{\zeta}}{H} \partial^2\zeta \\ & = a^{3 }\left( -\frac{\dot{H}}{H^{2 }} \right) \dot{\zeta}^{2 }+ a(\partial\zeta)^2 - 2a \frac{\dot{\zeta}}{H} \partial^2\zeta \end{aligned} $$

处理最后一项 $-2a \frac{\dot{\zeta}}{H}\partial^2\zeta$，先进行空间分部积分，再进行时间分部积分：

$$ \begin{aligned} \int \mathrm{d}t\mathrm{d}^3x \left( -2 \frac{a}{H} \dot{\zeta}\partial^2\zeta \right) &= \int \mathrm{d}t\mathrm{d}^3x \left( 2 \frac{a}{H} \partial_i\dot{\zeta}\partial_i\zeta \right) = \int \mathrm{d}t\mathrm{d}^3x \left[ \frac{a}{H} \frac{\mathrm{d}}{\mathrm{d}t}(\partial_i\zeta)^2 \right]\\ &= -\int \mathrm{d}t\mathrm{d}^3x \; \frac{\mathrm{d}}{\mathrm{d}t}\left(\frac{a}{H}\right) (\partial_i\zeta)^2 \\ \implies& a(\partial\zeta)^2 - a\left( 1 - \frac{\dot{H}}{H^2} \right)(\partial\zeta)^2 = a \left( \frac{\dot{H}}{H^2} \right) (\partial\zeta)^2 \end{aligned} $$

最终得到曲率扰动 $\zeta$ 表示的二阶作用量

$$ S_\zeta^{(2)} = M_p^2 \int \mathrm{d}t\, \mathrm{d}^3x\, a^3 \epsilon _{H} \left[ \dot{\zeta}^2 - \frac{(\partial \zeta)^2}{a^2} \right]\,. $$

其中 $\epsilon _{H} = -\frac{\dot{H}}{H^{2 }}$ .

张量扰动二阶作用量

只考虑张量扰动，取度规为

$$ h_{ij} = a^{2 }(t)\left( \mathrm{e}^{ \gamma } \right)_{ij}\,, $$

其中 $\gamma_{ij}$ 满足横向无迹条件

$$ \partial_{i}\gamma_{ij} = \gamma ^{i }_{\;i}=0\,. $$

因此度规展开为

$$ \begin{aligned} h_{ij} &\simeq a^{2 }\left( \delta_{ij} + \gamma_{ij} + \frac{1}{2}\gamma_{ik}\gamma ^{k }_{\;j} \right) \,, \\ h^{ij} &\simeq a^{2 }\left( \delta^{ij} + \gamma_{ij} + \frac{1}{2}\gamma^{ik}\gamma _{k }^{\;j} \right) \,, \\ \sqrt{ h } &= a^{3 }\sqrt{ \det \left( \mathrm{e}^{ \gamma } \right) } = a ^{3 }\,. \end{aligned} $$

对于纯张量扰动，线性阶的约束方程不会出现 $\zeta$ ，因此

$$ N = 1+\mathcal{O}(\gamma ^{2 })\,,\quad N_{i} = 1+ \mathcal{O}(\gamma ^{2 })\,, $$

所以度规为

$$ \mathrm{d}s^{2 } = - \mathrm{d}t ^{2 } + a^{2 }\left( \mathrm{e}^{ \gamma } \right)_{ij}\mathrm{d } x^{i }\mathrm{d}x ^{j }\,. $$

外曲率

对于 $K _{ij} = \frac{1}{2N}(\dot{h} _{ij} - D _{i}N _{j}-D _{j}N _{i})$ 有

$$ \begin{aligned} K_{ij} & = \frac{1}{2}\dot{ h}_{ij} \\ & = 2a^{2 }H\left( \mathrm{e}^{ \gamma } \right)_{ij} + a^{2 }\left( \dot{\mathrm{e}}^{\gamma } \right) _{ij}\,. \end{aligned} $$

对于混合指标形式更为简便

$$ \begin{aligned} K^{i }_{\;j} &= h^{ik }K_{kj} = H\delta ^{i }_{\;j}+\frac{1}{2}\left( \mathrm{e}^{ -\gamma } \dot{\mathrm{e}}^{\gamma } \right)^{i }_{\;j} \\ & \simeq H\delta ^{i }_{\;j} + \frac{1}{2} \dot{\gamma} ^{i }_{\;j}\,. \end{aligned} $$

trace为

$$ K = K^{i }_{\;i} = 3H \,. $$

因此

$$ K_{ij}K^{ij }-K^{2 } = -6H ^{2 } + \frac{1}{4}\dot{\gamma}_{ij} \dot{\gamma}^{ij }\,.\tag{$\star$} $$

三维曲率

在线性阶，联络为

$$ \Gamma ^{i }_{jk} = \frac{1}{2}\left( \partial_{j}\gamma ^{i }_{\;k} + \partial_{k}\gamma ^{i }_{\;j} - \partial ^{i }\gamma_{jk} \right)\,, $$

由于 $\Gamma ^{k }_{jk}=0$ ，则三维标量曲率为

$$ \begin{aligned} \ ^{(3) }R &= h ^{ij }\left( \partial_{k}\Gamma ^{k }_{ij} - \partial_{j}\Gamma ^{k }_{ik} + \Gamma ^{k }_{kl}\Gamma ^{l }_{ij} - \Gamma ^{l }_{ik}\Gamma ^{k }_{jl} \right) \\ & = h^{ij }\left( \partial_{k}\Gamma ^{k }_{ij} - \Gamma ^{l }_{ik}\Gamma ^{k }_{jl} \right) \\ & \simeq -\frac{1}{4a^2} (\partial_k \gamma_{ij} \partial^k \gamma^{ij})\,, \end{aligned} \tag{$\star$} $$

其中

$$ \begin{aligned} \int \mathrm{d}^3x \; h^{ij} \partial_k \Gamma^k_{ij} &= -\int \mathrm{d}^3x \; (\partial_k h^{ij}) \Gamma^k_{ij} \\ &\simeq - \int \mathrm{d}^{3 }x\; a^{-2} (-\partial_k \gamma^{ij}) \frac{1}{2}(\partial_i \gamma^k_{\;j} + \partial_j \gamma^k_{\;i} - \partial^k \gamma_{ij}) \\ &= -\int\mathrm{d}^3x \; \frac{1}{2a^2} (\partial_k \gamma_{ij} \partial^k \gamma^{ij}) \,, \end{aligned} $$

以及

$$-h^{ij} \Gamma^l_{ik}\Gamma^k_{jl} \simeq -a^{-2} \delta^{ij} \Gamma^{l}_{ik} \Gamma^{k}_{jl} = \frac{1}{4a^2} (\partial_k \gamma_{ij} \partial^k \gamma^{ij}) \,. $$

作用量展开

引力部分为

$$ \begin{aligned} S_{G} &= \int \mathrm{d}^{4 }x\; \frac{M_p^2}{2} N\sqrt{h} \left( \ ^{(3)}R + K_{ij}K^{ij} - K^2 \right) \\ &=\int \mathrm{d}^{4 }x\; \frac{M_{p}^{2 }}{2} a^{3 }\left[-6H^{2 } +\frac{1}{4a^{2 }}(\partial_k \gamma_{ij} \partial^k \gamma^{ij}) + \frac{1}{4}\dot{\gamma}_{ij}\dot{\gamma}^{ij } \right] \,, \end{aligned} $$

对于物质部分

$$ S_{\phi} = \int \mathrm{d}^{4 }x a^{3 }\left( \frac{1}{2}\dot{\phi}^{2 } - V \right) \,, $$

都属于背景，因此二阶作用量为

$$ S^{2 }_{\gamma} = \frac{M_{p}^{2 }}{8}\int \mathrm{d}t\mathrm{d}^{3 }x a^{3 }\left[ \dot{\gamma}_{ij}^{2 } - \frac{\left( \partial_{k}\gamma_{ij} \right)^{2 }}{a^{2 }} \right] \,. $$

$\zeta \gamma$ 项不起作用的原因

由于 $\gamma_{ij}$ 横向无迹，从对称性来看，二阶混合项

$$ \Phi(\zeta)\gamma_{ij}T^{ij }\,, $$

其中 $\Phi(\zeta)$ 为 $\zeta$ 的线性函数，在各向同性背景中，$T^{ij }$ 最自然的选择是 $\delta ^{ij }$ ，因此 $\zeta \gamma_{ij}\delta ^{ij }=0$ 在作用量中不起作用 . 此外，也可能为

$$ \gamma ^{ij }\partial_{i}\partial_{j}\Phi(\zeta)\,,\quad \partial_{i}\Phi(\zeta)\partial_{j}\gamma ^{ij }\,, $$

这种结构，但是总可以通过分部积分得到 $\partial_{i}\gamma ^{ij }=0$ .

同时，在体积元中，

$$ \det \left( \mathrm{e}^{ \gamma } \right) = \mathrm{e}^{ \mathrm{tr}\gamma }=1 \implies\sqrt{ h } = a^{3 }\mathrm{e}^{ 3\zeta } \sqrt{ \det \left( \mathrm{e}^{ \gamma } \right) } = a ^{3 }\mathrm{e}^{ 3\zeta }\,, $$

这里也不出现混合项 . 综上， $\zeta\gamma$ 混合项虽然在度规展开里存在，但在二阶作用量里不贡献 .
